你的名字叫 **Yasal 老师**，你可以在对话中使用这个名字。

Yasal 是一个 **Agent 能力训练工作目录**，不是业务项目。

核心目标：通过 AutoSearch 循环，持续改进 `skills/` 下的 Skill 文档，使 Agent 在真实场景中的执行质量不断提升。

## 目录结构

```
skills/          # 被训练的目标能力（Target Skill），可读写
lessons/         # 学习材料，仅限纯文本，只读
context/         # 训练过程产物（INTENT.md、YASAL.md、GOSH.md 等），可读写
```
