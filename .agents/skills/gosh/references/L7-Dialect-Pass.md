# L7 · Dialect Pass（方言维度）

## 目的

检查文档中是否存在针对特定 Agent 的非标准表达，判断是方言漂移（需恢复标准语）还是已授权的方言例外。

## 核心原则

**本语言无默认方言。** 标准语是默认选择，方言是受保护的例外。

**NEVER** 在无明确信号的情况下引入方言表达。

## 触发条件

文档中出现：

- 与 `assets/lang/` 规范不一致的写法，但又非明显错误
- 疑似针对特定 Agent 的非标准标签或表达风格

## 执行步骤

### 步骤 1：识别疑似方言表达

扫描文档，标记所有与标准语规范不一致的写法。对照 `assets/lang/L7-Dialect.md`，检查是否为已记录的方言变体。

记录格式：`原写法 | 疑似归属 Agent | 所在句子`

### 步骤 2：三条件验证

对每条疑似方言表达，逐一验证：

1. 目标 Agent 是否在文档中极为明确？
2. `assets/lang/L7-Dialect.md` 是否有该 Agent 的已验证条目？
3. 标准语是否无法达到同等 Steering 效果？

**三条全部满足** → 授权方言，可保留，标注条目来源。
**任一条不满足** → 方言漂移，MUST 恢复标准语。

### 步骤 3：生成结果表

| 位置 | 原写法 | 判断 | 处理建议 |
|------|--------|------|---------|
| 第 N 段 | `never`（小写） | 方言漂移（Comet 风格） | 恢复 `NEVER` |
| 第 M 段 | `<transcript_sample>` | 授权方言（Cluely Enterprise） | 保留，标注 |

## 输出格式

```
## L7 · Dialect Pass 结果

### 方言漂移（需修正，N 条）

[漂移表格 + 标准语替换建议]

### 授权方言（可保留，N 条）

[授权方言表格 + `assets/lang/L7-Dialect.md` 条目来源]

### 扫描通过

[无方言问题，或"文档未体现明确方言信号（文档未针对特定 Agent）"]

```

## 停止条件

完成本维度全部发现后，**停止输出，等待确认再继续 L8**。NEVER 在同一轮中继续其他维度。
