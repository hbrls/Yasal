# Aqoi

Aqoi 负责处理中性抑制表达。

它不负责主要改写，不负责分析业务，只负责在 `gosh` 之后做最后一轮静默清理和兜底。

## 核心任务

- 清理 `不要提及`、`不要写`、`无需说明`、`不必展开`、`避免出现` 等表达
- 消除“通过否定而再次点名目标”的上下文污染
- 在不损伤原意的前提下，把残留的反向中性表达静默去掉

## 与 gosh 的分工

- `gosh`：负责主要语言优化、结构改写、Steering 强化
- `aqoi`：负责最后清场，清掉会把被抑制对象重新写进文本的残留表达

默认顺序：先 `gosh`，后 `aqoi`。

## 明确的“不要提及”场景

- 中英文切换是正常现象，不把语言切换本身视为问题
- 默认状态的现象，不需要额外说明、辩护或强调
- 与 Steering 无关的正确性评判、风格标签或额外说明，应静默去掉
- 如果一句话的作用只是提醒“不要讨论某个中性对象”，而没有提供新的正向信息，该句应删除

## 与功能性反向约束的边界

`aqoi` 不删除所有否定表达。

以下情况保留：

- 反向约束承担真实的 Steering 功能
- 反向约束在划定有效边界，而不是重复点名一个中性对象
- 反向约束同时给出正向替代方向，例如 `not X, but Y`

换句话说，`aqoi` 清理的是“中性对象被再次点名”的残留表达，不清理真正有功能的边界约束。

## When Activated

以下任一情况出现时，调用 `aqoi`：

- 文本已经过 `gosh` 处理，但仍残留“不要提及”类表达
- 需要做最终版清理，避免被抑制对象继续出现在上下文中
- 某些句子的作用只是提醒“不要说某事”，而不是提供正向信息
- 文本对中性现象做了多余说明，例如解释某现象“本来就正常”
- 文本对默认状态做了防御性表述、辩护性表述或自我解释

## 执行步骤

1. 读取待清理片段，默认假定主要语言问题已由 `gosh` 先处理。
2. 扫描所有会显式点名被抑制对象的表达，例如：
   - `不要提及 ...`
   - `不要写 ...`
   - `无需说明 ...`
   - `不必展开 ...`
   - `避免出现 ...`
   - `不把 ... 视为问题`
   - `这是正常现象`
   - `不需要额外说明 / 辩护 / 强调`
3. 对每处命中，先判断它属于哪一类：
   - 中性对象的重复点名
   - 默认状态的多余解释
   - 真正有功能的反向约束
4. 对属于前两类的命中，按以下优先级处理：
   - 删除后不影响语义：直接删除
   - 该句只有提醒作用：整句删除
   - 需要保留约束效果：改写成不点名目标的正向表达
5. 对属于“真正有功能的反向约束”的命中，保留不动。
6. 清理后再次通读，确认输出中不再残留被清理对象的显式提及，也没有误删功能性边界。
7. 只输出清理后的文本，默认不解释清理过程。

## 改写原则

- 优先删除，不优先否定
- 优先正向目标，不重复被抑制对象
- 优先静默完成，不额外宣告“已清理”
- 若删除会造成语义断裂，则做最小正向改写
- 中性现象默认不立案，不为其补写解释
- 默认状态无需辩护，不为其追加元说明

## 典型可删句型表

| 句型 | 默认处理 |
|------|----------|
| `不要提及 X` | 删除整句，或删除 `不要提及 X` 后保留主句 |
| `无需说明 X` | 直接删除 |
| `不必展开 X` | 直接删除 |
| `避免出现 X` | 若只是提醒，直接删除；若需保留效果，改写成正向要求 |
| `X 是正常现象` | 直接删除 |
| `不把 X 视为问题` | 直接删除 |
| `不需要额外说明 / 辩护 / 强调` | 直接删除 |
| `我们这里不讨论 X` | 若无新增信息，整句删除 |
| `无需解释为什么不做 X` | 直接删除 |

## 简例

原文：

```text
不要提及语言混用是否自然，直接进入语法失效分析。
```

处理后：

```text
直接进入语法失效分析。
```

原文：

```text
中英文切换是正常现象，不把语言切换本身视为问题。直接检查是否存在无功能切换。
```

处理后：

```text
直接检查是否存在无功能切换。
```

原文：

```text
无需说明为什么不讨论这一点，直接给结果。
```

处理后：

```text
直接给结果。
```

原文：

```text
Your goal is to create materials with genuine learning value — not generic summaries, but deeply organized knowledge.
```

处理后：

```text
Your goal is to create materials with genuine learning value — not generic summaries, but deeply organized knowledge.
```
